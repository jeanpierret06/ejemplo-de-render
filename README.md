# 🚀 Práctica 1: Formulario Dinámico y Despliegue en la Nube

¡Bienvenido! Este repositorio contiene un proyecto web dinámico diseñado como una práctica interactiva para dominar la manipulación de formularios HTML/PHP, la aplicación de estilos modernos con Bootstrap 5 y los flujos de integración y despliegue continuo (CI/CD) en plataformas Cloud como **Render** o **Railway**.

---

## 📋 Características del Proyecto

*   **Interfaz de Usuario Moderna:** Vista web optimizada con Bootstrap 5 que incluye un diseño de tarjeta (`card`) centrado y responsivo para una experiencia fluida tanto en computadoras como en dispositivos móviles.
*   **Procesamiento de Datos:** Un flujo de datos estructurado que captura la información del usuario mediante el método `POST` y la transfiere a un archivo de procesamiento backend (`logica.php`).
*   **Preparado para Producción:** Configuración ligera y limpia ideal para subir a entornos PaaS (Platform as a Service) sin configuraciones pesadas de servidores.

---

## 🛠️ Tecnologías Utilizadas

*   **HTML5 & CSS3:** Estructura semántica con estilos personalizados embebidos para agilizar la carga.
*   **Bootstrap v5.3:** Framework de diseño para componentes responsivos y tipografía limpia.
*   **PHP:** Procesamiento lógico de las peticiones del lado del servidor.
